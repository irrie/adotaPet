# adotaPet
website para resgate e adoção de animais

# Tutorial
1 - Fazer o clone do projeto\
2 - Importar o banco de dados mysql\
        mysql -u root -p\
        mysql> CREATE DATABASE adocao_site;\
        mysql> exit;\
        mysql -u seu_nome_de_usuario -p adocao_site < data-dump.sql\
3 - Adicionar sua senha do mysql ao server.js\
4 - Escrever cd Adocao-Site no terminal\
5 - Rodar npm start no terminal\
